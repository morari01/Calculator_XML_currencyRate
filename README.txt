
Poniżej znajduje opis programu Kalkulator walut.

Żeby uruchomić program: 

1. Otworzyć środowisko programistyczne Intellij Idea lub Eclipse lub inne.
2. Uruсhomiс projekt (przycisk RUN).
Program napisany w języku Java z użyciem Java Swing i nie potrzebuje instalacji dodatkowych bibliotek java.

Jak dziala program: 
Po uruchamieniu program prosi o podaniu kwoty w walucie EURO.
Przy podaniu kwoty w postaci Tekstu - wystąpi bląd. 
Póżnej z listy proponowanych walut obieramy tą, ktorą potrzebujemy.
Nacziskamy przycisk "Oblicz kwotę".
Program wyswietla kwotę oblicząną z EURO w obranej walucie.
Można obliczać różne walute kilka razy.
Zamykamy okno.
